Project title and description:

Fleure is a simple and minimalist flower shop e-commerce that was created for individual assignment (IMS566). The website was designed to provide smooth shopping experiences while meeting the functional requirements of e-commerce basic online store. The system allow users to login, browse flowers by category, add items to the cart, checkout then view order confirmation.

Features included:

1) Start with login or authentication page where there are simple login system and logout function.
2) Dashboard page - there are bestsellers items, customer's review, product distribution category chart, quick actions and footer that shows Fleure's slogan, social media and copyright holder with minimalist design.
3) Products system - user can view products by category such as Roses, Tulips, LiLies, Orchids, Mixed Flowers and Sunflowers. Also products cards include with image, price, stock availability, quantity selector, remarks or notice of pickup and delivery and add to cart function.
4) Cart and Checkout - view and remove selected items, can increase or decrease quantity that limited by stock, auto-calculated sub-total and total, users can checkout and can proceed to order confirmation page.
5) About us - background story of Fleure, vision and mission, and sales chart with the explanation beside the chart.

Overall with smooth navigation with responsive layout.

Frameworks/libraries:

1) Front-end layout, grids, button, cards and icons use Bootstrap 5 and Bootstrap icons
2) Google Fonts which is BBH Sans Hegarty
3) For sales chart is Chart.js
4) Back-end - use PHP for pages, rooting and login authentication
5) Javascript for logic
6) SessionStorage use to store cart items

There are also other libraries such as HTML5 and CSS to style the system
